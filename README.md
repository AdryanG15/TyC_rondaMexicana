# Documentos legales - La Ronda Mexicana

Este directorio contiene los documentos legales de la app para enlazar en Google Play, App Store y la propia aplicación.

## Archivos

| Documento | Archivo | Uso |
|-----------|---------|-----|
| Política de Privacidad | `politica-privacidad.html` | Google Play, App Store, App |
| Términos y Condiciones | `terminos-y-condiciones.html` | Google Play, App Store, App |

## URLs en GitHub Pages

Si tienes el repositorio en GitHub y activas **GitHub Pages** desde la carpeta `docs`:

1. Ve a tu repositorio en GitHub
2. **Settings** → **Pages**
3. En "Source" elige **Deploy from a branch**
4. Branch: `main` (o `master`)
5. Folder: **/docs**
6. Guarda

Las URLs serán (reemplaza `TU-USUARIO` y `TU-REPO`):

- **Política de Privacidad:**  
  `https://TU-USUARIO.github.io/TU-REPO/legal/politica-privacidad.html`

- **Términos y Condiciones:**  
  `https://TU-USUARIO.github.io/TU-REPO/legal/terminos-y-condiciones.html`

### Ejemplo

Si tu repo es `https://github.com/adrian1515/rondaMexicana`:

- Política: `https://adrian1515.github.io/rondaMexicana/legal/politica-privacidad.html`
- Términos: `https://adrian1515.github.io/rondaMexicana/legal/terminos-y-condiciones.html`

## Dónde enlazar

- **Google Play Console:** Ficha de la tienda → Política de privacidad (URL obligatoria)
- **App Store Connect:** Información de la app → URL de política de privacidad
- **En la app:** Pantalla de usuario, configuración o "Acerca de" (opcional)
